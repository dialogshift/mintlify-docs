# Tracking mit URL-Parametern

DialogShift markiert weitergeleiteten Traffic, um Buchungen aus dem Chatbot in Ihrer Booking Engine und Analytics nachzuvollziehen.

<Warning>
**Wichtiger Hinweis (Mai 2024):** Google hat die Auswertung von UTM-Parametern verändert. Ein zuverlässiges E-Commerce Tracking über Chatbot UTM-Parameter ist nicht mehr möglich. Nutzen Sie stattdessen den `ref`-Parameter.
</Warning>

## Tracking-Methoden

### 1. URL-Parameter

DialogShift fügt automatisch folgenden Parameter hinzu:

```
&ref=dialogshift
```

### 2. Referrer-Domain

Als Alternative kann die Referrer-Domain genutzt werden:
- Domain: `dialogshift.com`
- Einschränkung: Qualitative Abstriche möglich

## Booking Engine Statistiken

<Steps>
  <Step title="Statistik öffnen">
    Öffnen Sie die Statistik Ihrer Web Booking Engine (WBE)
  </Step>
  <Step title="Traffic-Herkunft prüfen">
    Suchen Sie nach Aufschlüsselung nach Traffic-Quelle
  </Step>
  <Step title="Chatbot-Traffic identifizieren">
    Filtern Sie nach `ref=dialogshift` oder Referrer `dialogshift.com`
  </Step>
</Steps>

## Google Analytics Integration

### Custom Report erstellen

<Tabs>
  <Tab title="Universal Analytics">
    1. Erstellen Sie einen Custom Report
    2. Filtern Sie E-Commerce Umsätze
    3. Bedingung: URL enthält `ref=dialogshift`
  </Tab>
  <Tab title="GA4">
    1. Öffnen Sie Explorative Datenanalyse
    2. Erstellen Sie einen Filter für Page Path
    3. Bedingung: enthält `ref=dialogshift`
  </Tab>
</Tabs>

<Note>
Die genaue Implementierung hängt von Ihrem E-Commerce Tracking Setup ab. Sprechen Sie mit Ihrem Analytics-Dienstleister.
</Note>

## Kanalbeschränkungen

<Info>
Aus technischen Gründen funktioniert URL-Parameter Tracking nicht bei:
- Instagram
- Facebook Messenger
- Google's Business Messages
- WhatsApp

Nutzen Sie für diese Kanäle das Tracking Pixel für indikative Zahlen.
</Info>

## Best Practices

<AccordionGroup>
  <Accordion title="WBE-Anbieter kontaktieren">

  </Accordion>
  <Accordion title="Kombiniertes Tracking">

  <Accordion title="Regelmäßige Überprüfung">
