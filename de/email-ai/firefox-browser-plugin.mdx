# DialogShift Browser Plugin für Firefox

Nutzen Sie die Email AI Funktionen flexibel im Firefox-Browser für die Kommunikation auf OTA-Portalen.

## Installation

<Steps>
  <Step title="Plugin herunterladen">
    Öffnen Sie den Installationslink in Firefox:
    
    <Card title="Firefox Plugin v1.1.1" icon="firefox" href="https://assets.dialogshift.com/emailai/firefox/d858e6baca08499cbe96-1.1.1.xpi">
      Plugin installieren
    </Card>
  </Step>
  
  <Step title="Installation erlauben">
    Bestätigen Sie die Installation in den erscheinenden Pop-ups
  </Step>
  
  <Step title="Button platzieren">
    Fügen Sie den Plugin-Button zur Firefox-Symbolleiste hinzu
  </Step>
</Steps>

## Nutzung

<CardGroup cols={2}>
  <Card title="1. Anmelden" icon="user">
    Mit DialogShift-Login authentifizieren
  </Card>
  <Card title="2. Nachricht einfügen" icon="message">
    Gast-Nachricht in das Textfeld kopieren
  </Card>
  <Card title="3. Antwort generieren" icon="sparkles">
    AI-Antwort erstellen und anpassen
  </Card>
  <Card title="4. Automatik nutzen" icon="wand-magic-sparkles">
    Wo möglich, liest das Plugin Nachrichten automatisch aus
  </Card>
</CardGroup>

<Tip>
Nutzen Sie Instruktionen, um die generierten Antworten nach Ihren Wünschen anzupassen.
</Tip>

<Info>
Das Plugin ermöglicht die direkte Nutzung der Email AI Funktionen in verschiedenen Chat-Umgebungen der OTA-Portale.
</Info>
