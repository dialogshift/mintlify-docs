# WhatsApp Telefonnummer auswählen

## Grundsätzliche Möglichkeiten

<CardGroup cols={2}>
  <Card title="Hauptnummer" icon="building">
    Viele Hotels nutzen ihre Hauptnummer auch für WhatsApp
  </Card>
  <Card title="Separate Nummer" icon="mobile">
    Eigene Mobilfunknummer (oft kostenlos vom Anbieter)
  </Card>
</CardGroup>

## Wichtige Überlegungen

### 1. Bereits genutzte Nummern

<Warning>
**Nummer bereits in Verwendung?**
Wenn die Nummer bereits mit einem WhatsApp Konto verbunden ist (privat oder Business App), muss sie VOR dem Setup getrennt werden. **Alle Daten gehen dabei verloren!**
</Warning>

<Card title="Anleitung: Nummer trennen" icon="unlink" href="/de/chat-ai/whatsapp/nummer-trennen">
  So trennen Sie eine Telefonnummer von einem bestehenden WhatsApp Konto
</Card>

### 2. Verifizierungsanforderungen

Die Telefonnummer wird von WhatsApp verifiziert:

<Tabs>
  <Tab title="SMS-fähige Nummer">
    - Verifizierungscode wird per SMS verschickt
    - Einfachster und schnellster Weg
    - Keine Terminvereinbarung nötig
  </Tab>
  <Tab title="Nicht SMS-fähige Nummer">
    - Verifizierung per Anruf in 10-Minuten-Fenster
    - Englische Stimme liest 6-stelligen Code vor
    - IVR muss dafür deaktiviert werden
    - Termin muss vereinbart werden
  </Tab>
</Tabs>

<Tip>
Wählen Sie wenn möglich eine SMS-fähige Nummer für einen reibungsloseren Verifizierungsprozess.
</Tip>
