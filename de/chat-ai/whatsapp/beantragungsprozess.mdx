# WhatsApp Beantragungsprozess

Die Aktivierung von WhatsApp Business API erfolgt in mehreren Schritten. DialogShift begleitet Sie durch den gesamten Prozess.

## Übersicht des Prozesses

<Steps>
  <Step title="Vorbereitung">
    Informationen sammeln und Business Manager vorbereiten
  </Step>
  <Step title="Antrag einreichen">
    Formular mit Basisdaten ausfüllen
  </Step>
  <Step title="Business-Verifizierung">
    Meta überprüft Ihr Unternehmen
  </Step>
  <Step title="Nummer aktivieren">
    Telefonnummer verifizieren und freischalten
  </Step>
</Steps>

## 1. Vorbereitung

<Card title="Checkliste" icon="clipboard-check">
  - Meta Business Manager Account
  - Handelsregisterauszug
  - Offizielle Telefonnummer
  - Unternehmenswebsite
  - Logo (min. 640x640px)
</Card>

## 2. Antrag einreichen

<Card title="DialogShift Onboarding-Formular" icon="form" href="https://forms.gle/VgtgqL5NK12Sryti6">
  Starten Sie hier Ihren WhatsApp-Antrag
</Card>

### Benötigte Informationen:
- **Facebook Business Manager ID** ([So finden Sie Ihre ID](https://www.tyntec.com/docs/faq-whatsapp-business-onboarding-how-can-i-find-my-facebook-business-manager-id))
- **Rechtlicher Unternehmensname** (wie im Handelsregister)
- **WABA Name** (Anzeigename für WhatsApp)
- **Telefonnummer** (im internationalen Format: +49...)
- **Unternehmensprofil** (Beschreibung, Adresse, Website)

<Warning>
Alle Angaben müssen mit Ihren offiziellen Dokumenten übereinstimmen!
</Warning>

## 3. Business-Verifizierung

Nach Einreichung prüft Meta Ihr Unternehmen. Dies kann 1-5 Werktage dauern.

<Info>
**Bereits verifiziert?** Wenn Ihr Business Manager bereits für andere Meta-Services verifiziert ist, entfällt dieser Schritt.
</Info>

Details zur Verifizierung: [Meta Unternehmensverifizierung →](/de/chat-ai/whatsapp/meta-unternehmensverifizierung)

## 4. Telefonnummer aktivieren

Nach erfolgreicher Prüfung erhalten Sie den Status "Ready for activation".

### Verifizierungsoptionen:

<Tabs>
  <Tab title="SMS (empfohlen)">
    **Ablauf:**
    1. Sie erhalten einen 6-stelligen Code per SMS
    2. Code an onboarding@tyntec.com weiterleiten
    3. Aktivierung innerhalb weniger Stunden
    
    **Vorteile:**
    - Schnellste Option
    - Kein Termin nötig
    - Automatischer Prozess
  </Tab>
  
  <Tab title="Sprachanruf">
    **Ablauf:**
    1. [Termin vereinbaren](https://www.tyntec.com/form-whatsapp-schedule-whatsapp-phone-number-verification)
    2. Automatischer Anruf zur vereinbarten Zeit
    3. Englische Ansage mit 6-stelligem Code
    4. Code notieren und übermitteln
    
    **Wichtig:**
    - IVR/Anrufbeantworter deaktivieren
    - Person muss Anruf entgegennehmen
    - 10-Minuten-Zeitfenster
  </Tab>
</Tabs>

## 5. Finaler Go-Live

Nach erfolgreicher Verifizierung:
- DialogShift konfiguriert Ihren Account
- Integration in Ihre DialogShift-Umgebung
- Testphase und Schulung
- Produktivschaltung

<Tip>
**Gesamtdauer:** Der komplette Prozess dauert in der Regel 3-7 Werktage, abhängig von der Business-Verifizierung.
</Tip>

## Häufige Probleme

<AccordionGroup>
  <Accordion title="Verifizierung abgelehnt">
    - Prüfen Sie die Übereinstimmung aller Daten
    - Stellen Sie sicher, dass Ihre Website vollständig ist
    - Handelsregisterauszug muss aktuell sein
  </Accordion>
  
  <Accordion title="SMS kommt nicht an">
    - Nummer im internationalen Format angeben
    - SMS-Empfang muss möglich sein
    - Ggf. auf Sprachanruf wechseln
  </Accordion>
  
  <Accordion title="Nummer bereits in Verwendung">
    Die Nummer muss zuerst von WhatsApp getrennt werden.
    [Anleitung →](/de/chat-ai/whatsapp/nummer-trennen)
  </Accordion>
</AccordionGroup>
