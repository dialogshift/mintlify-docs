# Live Chats beantworten

Wenn die Live Chat Option aktiviert ist, können Sie direkte Nachrichten von Gästen empfangen.

<Info>
Live Chats entstehen wenn:
- Der Chatbot keine passende Antwort findet
- Ein Besucher manuell den Live Chat auswählt
</Info>

## Zugriff auf Live Chats

<Steps>
  <Step title="Navigation">
    Klicken Sie auf **Live Chat** im Seitenmenü
  </Step>
  <Step title="Nachricht auswählen">
    Wählen Sie unter **Offen** die zu beantwortende Nachricht
  </Step>
  <Step title="Chat abschließen">
    Nach der Antwort klicken Sie auf **Chat parken** oder **Chat beenden**
  </Step>
</Steps>

## Chat-Status verstehen

<CardGroup cols={3}>
  <Card title="Offen" icon="inbox">
    Nachrichten die auf Antwort warten
  </Card>
  <Card title="Geparkt" icon="pause">
    Chat pausiert aber noch aktiv
  </Card>
  <Card title="Beendet" icon="check">
    Chat geschlossen, Chatbot übernimmt wieder
  </Card>
</CardGroup>

<Warning>
**Geparkte Chats** sind weiterhin im Live-Chat-Modus:
- Bei neuer Nachricht → automatisch wieder **Offen**
- Nach 24h Inaktivität → automatisch **Beendet**
</Warning>

## Zusätzliche Funktionen

- **Zuweisungen:** Chats können bestimmten Inboxen oder Personen zugewiesen werden (oberes Chatfenster)
- **Textbausteine:** Vorgefertigte Antworten über die drei Punkte im unteren Chatfenster erstellen
