---
title: "Passwort zurücksetzen"
description: "So setzt Du Dein Passwort zurück."
---

Bevor Du Dein Passwort zurücksetzen kannst musst Du aus der DialogShift App ausgeloggt sein.

1. Navigiere zu: https://app.dialogshift.com/login
2. Klicke auf "Passwort vergessen"
3. Gib deinen Usernamen der gleich ist mit Deiner E-Mail Adresse ein
4. Unser System sendet Dir via E-Mail einen 6-stelligen Code (schaue auch im Spam Folder nach, wenn Du ihn nicht sofort findest)
5. Gib den Code zusammen mit einem neuen Passwort ein
6. Logge Dich mit Usernamen und dem neuen Passwort ein.
