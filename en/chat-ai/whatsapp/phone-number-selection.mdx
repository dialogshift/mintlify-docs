Choosing the right phone number is an important step for your WhatsApp Business account.

## Available Options

<CardGroup cols={2}>
  <Card title="Main Number" icon="building">
    **Use hotel main number**
    - Guests already know this
    - Unified communication
    - Note: No normal telephony possible anymore
  </Card>
  <Card title="New Number" icon="phone">
    **Set up separate number**
    - Separation of telephony and WhatsApp
    - More flexible use
    - Often free from provider
  </Card>
</CardGroup>

## Technical Requirements

### Supported Number Types:
- ✅ **Mobile numbers** (recommended)
- ✅ **Landline numbers**
- ✅ **Toll-free numbers** (0800, etc.)
- ✅ **International numbers**
- ❌ **Short numbers** (not supported)

### Important Restrictions:

<Warning>
**Number already in WhatsApp?**  
Numbers already connected to WhatsApp or WhatsApp Business App must FIRST be disconnected.
[Guide: Disconnect number →](/en/chat-ai/whatsapp/disconnect-number)
</Warning>

<Info>
**One number per location**  
Each physical hotel location needs its own WhatsApp number. One number for multiple hotels is not allowed.
</Info>

## Verification Process

The chosen number must be verified:

<Tabs>
  <Tab title="SMS Verification ⭐">
    **Recommended for:**
    - Mobile numbers
    - SMS-capable landline numbers
    
    **Advantages:**
    - Immediate verification
    - No appointment needed
    - Automatic process
    
    **Process:**
    1. Receive code via SMS
    2. Forward to DialogShift
    3. Automatic activation
  </Tab>
  
  <Tab title="Call Verification">
    **Necessary for:**
    - Pure landline numbers
    - Numbers without SMS reception
    
    **Note:**
    - Appointment required
    - Disable IVR/answering machine
    - Person must be available
    - English announcement
    
    **Process:**
    1. Schedule appointment
    2. Answer call
    3. Note 6-digit code
  </Tab>
</Tabs>

## Best Practices

<Steps>
  <Step title="Check availability">
    Make sure the number is not already used for WhatsApp
  </Step>
  <Step title="Choose SMS-capable">
    SMS-capable numbers significantly speed up the verification process
  </Step>
  <Step title="Inform your team">
    When using main number: inform team about changes
  </Step>
  <Step title="Plan ahead">
    For landline numbers: consider IVR shutdown and appointment scheduling
  </Step>
</Steps>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Can I change my number later?">
    Yes, but changing requires new verification and all guests must be informed about the new number.
  </Accordion>
  
  <Accordion title="What happens to incoming calls?">
    When using your main number for WhatsApp, normal phone calls on this number are no longer possible.
  </Accordion>
  
  <Accordion title="Can I use a foreign number?">
    Yes, international numbers are possible. Note that guests may pay international fees.
  </Accordion>
</AccordionGroup>

<Tip>
**Recommendation:** Use a separate mobile number for maximum flexibility and quick setup.
</Tip>
