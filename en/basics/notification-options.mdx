# Notification Options

DialogShift offers various notification options that can be configured both in the user profile and across hotels.

## User Profile Settings

For your own user, notifications can be activated under [Organization >> Notifications](https://app.dialogshift.com/notifications).

### 1. Email Notifications

Activation for the email address you signed in with.

<Note>
  Additional email addresses can only be added across hotels (see below).
</Note>

### 2. Smartphone / Mobile Device

Push notifications via the DialogShift Mobile App.
