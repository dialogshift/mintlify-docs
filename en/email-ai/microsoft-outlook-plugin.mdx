## Requirements

To use the Email AI plugin in Microsoft Outlook, the following requirements must be met:

<CardGroup cols={2}>
  <Card title="DialogShift Account" icon="user">
    User must be registered with DialogShift
  </Card>
  <Card title="Exchange Server" icon="server">
    Exchange Server required (Exchange Online or Exchange 2016+)
  </Card>
</CardGroup>

<Note>
IMAP accounts are not supported.
</Note>

### Compatible Outlook Versions

<Tabs>
  <Tab title="Windows">
    **Supported:**
    - Outlook 2021 or newer
    - Outlook 365 on Windows 10 20H2 or newer
    - New Outlook

    **NOT supported:**
    - Outlook 2019 or older
    - Outlook 365 (not regularly updated or on Windows 10 2004 or older)
  </Tab>
  
  <Tab title="Mac">
    **Supported:**
    - Outlook App & Outlook 2024

    **NOT supported:**
    - Outlook with year number (except 2024)
  </Tab>
  
  <Tab title="Web">
    **Supported:**
    - Outlook online (Firefox, Safari, Chrome & derivatives)

    **NOT supported:**
    - Outlook Web Access
  </Tab>
</Tabs>

## Installation

### With Exchange Online

<Steps>
  <Step title="Open Outlook">
    Start Outlook on your computer
  </Step>
  
  <Step title="Open Apps menu">
    Click "All Apps" in the top right and select "Get Add-ins"
  </Step>
  
  <Step title="Search for plugin">
    Enter "DialogShift" in the search field at the top right
  </Step>
  
  <Step title="Install plugin">
    Select "DialogShift Email Assistant" and click "Add"
  </Step>
  
  <Step title="Use plugin">
    You can find the plugin in the "Add-ins" tab under "My Add-ins"
  </Step>
</Steps>

### With Exchange On-Premise

<Steps>
  <Step title="Open Outlook Web Access">
    Start the Outlook web interface in your browser and sign in with an account that has sufficient permissions
  </Step>
  
  <Step title="Open Apps overview">
    Navigate to the overview of all installed apps
  </Step>
  
  <Step title="Add plugin via URL">
    Select the option to add a new plugin via URL and enter the following URL:
    
    ```
    https://store.office.com/app/download?assetid=WA200006927&cmu=en-001&ClientSessionId=9e50467e-3598-4440-b754-797ccc61f5b1
    ```
  </Step>
</Steps>
