# DialogShift Browser Plugin for Firefox

Use Email AI functions flexibly in the Firefox browser for communication on OTA portals.

## Installation

<Steps>
  <Step title="Download plugin">
    Open the installation link in Firefox:
    
    <Card title="Firefox Plugin v1.1.1" icon="firefox" href="https://assets.dialogshift.com/emailai/firefox/d858e6baca08499cbe96-1.1.1.xpi">
      Install plugin
    </Card>
  </Step>
  
  <Step title="Allow installation">
    Confirm the installation in the appearing pop-ups
  </Step>
  
  <Step title="Place button">
    Add the plugin button to the Firefox toolbar
  </Step>
</Steps>

## Usage

<CardGroup cols={2}>
  <Card title="1. Sign in" icon="user">
    Authenticate with DialogShift login
  </Card>
  <Card title="2. Insert message" icon="message">
    Copy guest message into the text field
  </Card>
  <Card title="3. Generate response" icon="sparkles">
    Create and customize AI response
  </Card>
  <Card title="4. Use automation" icon="wand-magic-sparkles">
    Where possible, the plugin automatically reads messages
  </Card>
</CardGroup>

<Tip>
Use instructions to customize generated responses according to your preferences.
</Tip>

<Info>
The plugin enables direct use of Email AI functions in various chat environments of OTA portals.
